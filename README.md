# CalorieTracker Client
CalorieTracker.Client is the ui for the Calorie Tracker application. It allows users to register and log in, add food entries by barcode, and view their nutrition statistics on a daily, weekly, and monthly basis.

## Requirements
- Node.js v24.14.1
- The [CalorieTracker.Api](https://github.com/vharudka/CalorieTracker/tree/master) backend running locally

## How to run
Before starting the UI, install dependencies and build the project:
```
npm install
npm run build
```
Then start the development server:
```
npm run dev
```
The UI will be available at:
```
http://localhost:5173/login
```
